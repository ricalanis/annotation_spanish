# Code for Poster 
## Towards testing effectiveness of a Spanish Sarcasm Model in different Regions

## Introduction

This repo contains the code and the data used for poster. It includes 

* Notebooks for Processing the data
* Source Data Used
* Intermediary Data
* Original Annotation data and processed data

## Libraries used

This repo uses

* Python
* Pandas
* Pickle
* Scikit learn for some metrics
